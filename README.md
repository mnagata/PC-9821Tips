#### HSB.EXEのPentium対応方法
HSB for PC-9801  
https://www.vector.co.jp/soft/dos/util/se002233.html
```
バイナリエディタで D5,08,81,E1,D5,08,3B,C1 と並んでいる直後の74,xxを90,90に変更
```

https://www.iodata.jp/lib/software/m/355.htm#MS-DOS
```

```
